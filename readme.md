    微信小程序仿皮皮虾小视频后台（功能持续更新中）
    
    # 主要功能
    - 用户登陆(token方式)注册
    - 用户上传头像
    - 用户上传视频
    - 视频列表浏览播放
    - 后续还会添加点赞、评论、热搜等功能
    # 技术框架
    - 核心框架：Springboot 2.x
    - 缓存框架：Redis 4.0
    - 持久层框架：MyBatis 3 tk.mybatis 1.2.4 pagehelper 1.2.3
    - 数据库连接池：HikariCP
    - 日志管理：Log4j
    - 其他：lombok 1.18.8 jsr303 swagger2 2.4.0
    # 开发环境
    - IDE：idea
    - DB：MySQL 5.7
    - JDK：1.8
